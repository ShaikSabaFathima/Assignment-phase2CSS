The Font-Family property

The font-family property specifies a list of fonts, from highest priority to lowest.

Font selection does not stop at the first font in the list that is on the user's system.

The Font-Size Property

medium Sets the font-size to a medium size.

x-small Sets the font-size to an xx-small size

xx-large Sets the font-size to an xx-large size

The Font-Weight Property

normal Defines normal characters

bold Defines thick characters

lighter Defines lighter characters
